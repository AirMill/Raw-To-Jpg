# RAW → JPG

Batch convert RAW photos to high-quality JPG files with a simple and efficient interface.

Designed for photographers and high-volume workflows.

---

## ✨ Features

- Batch RAW conversion
- Supports major formats (Nikon, Canon, Sony, etc.)
- Fast processing
- Clean and distraction-free UI
- Local processing (your files stay on your computer)
